To use this board:

Install the USB drivers for the SparkFun boards, following the steps for Windows or for Linux and Mac. In the Arduino IDE go to File->Preferences->Additional Boards Manager URLs and add: https://raw.githubusercontent.com/sparkfun/Arduino_Boards/master/IDE_Board_Manager/package_sparkfun_index.json Go to Tools->Board->Boards Manager, search for SparkFun AVR Boards and click on install. Go to Tools->Board and select SparkFun Pro Micro. Go to Tools->Processor and select ATmega32U4 (3.3V, 8MHz).

Uploading Firmware

‍The serial monitor must be closed before uploading code to the device.

  1. Switch the KISS LoRa device off.
  2. Plug it into your computer using a micro USB cable.
  3. In the Arduino IDE make sure the correct Tools->Board (SparkFun Pro Micro) and Tools->Processor (ATmega 32U4, 8MHz) are selected.
  4. Switch the device on and make sure the correct Tools->Port is selected.
  5. Switch the device off again.
  6. Press and hold the Push Button on the kiss device.
  7. In the Arduino IDE, choose Sketch->Upload.
  8. As soon as you see PORTS {} / {} => {} printed in the output window turn KISSLoRa on.
  9. When the upload process continue past showing PORTS {} / {} => {}, you can release the push button.
